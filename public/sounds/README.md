# Sonidos de Animales

Esta carpeta contiene los archivos de audio para los sonidos de los animales.

## Cómo obtener los sonidos

Puedes descargar sonidos gratuitos de animales desde Pixabay:
- Visita: https://pixabay.com/sound-effects/search/animal/

## Archivos necesarios

Descarga los siguientes archivos y guárdalos en esta carpeta con estos nombres:

- `rabbit.mp3` - Sonido de conejo (suave y amigable)
- `bear.mp3` - Gruñido de oso (moderado, no muy fuerte)
- `fox.mp3` - Ladrido/lloriqueo de zorro
- `frog.mp3` - Croar de rana
- `squirrel.mp3` - Chirrido de ardilla
- `owl.mp3` - Ulular de búho
- `wolf.mp3` - Aullido de lobo
- `deer.mp3` - Bramido de ciervo
- `bee.mp3` - Zumbido de abeja
- `butterfly.mp3` - Sonido suave de mariposa (alas)
- `ladybug.mp3` - Sonido suave de mariquita
- `lizard.mp3` - Siseo suave de lagarto

## Recomendaciones

1. **Duración**: Los sonidos deben ser cortos (1-3 segundos) para mantener el juego fluido
2. **Volumen**: Asegúrate de que los sonidos no sean demasiado fuertes (el código ajusta el volumen a 0.7)
3. **Formato**: Usa formato MP3 para mejor compatibilidad
4. **Calidad**: Sonidos claros y amigables para niños
